# My easy to go calculator

My easy to go calculator is for everyone. It adds, subtract,divide, multiply, and gives power and remainder of two numbers.

you have to clone this repository with

> git clone [github.com/bihakhan1-spec/Myeasytogocalculator.git](https://github.com/bihakhan1-spec/Myeasytogocalculator.git)
>

## ✨ What it does

- Takes **two numbers** from the user
- Calculates and prints:
  - ➕ Addition
  - ➖ Subtraction
  - ✖️ Multiplication
  - ➗ Division (rounded to 2 decimals)
  - 🔢 Power (`**`)
  - 📐 Remainder (`%`)

---

## 🧠 Concepts I used

- `input()` to take user input
- **Type casting** (`float()`) to turn text into numbers
- **Variables** to store values
- **Arithmetic operators** (`+ - * / ** %`)
- **f-strings** for clean, formatted output
